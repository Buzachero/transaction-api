# Asynchronous Bank API with FastAPI

This RESTful API was developed to be asynchronous using FastAPI. It handles deposit and withdrawal operations on bank accounts.

## Features

- **Accounts Registration:** It allows bank accounts registration.
- **Accounts List:** It shows all bank accounts registered.
- **Transactions Registration:** It allows bank transactions registration, like deposit and withdrawal.
- **Bank Statement:** It shows all transactions given an account.
- **JWT Authentication:** It uses JWT to ensure only authenticated users are allowed to access the endpoints.
